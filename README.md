# calculator-boilerplate
A boilerplate bootstrap/node.js project, with a calculator example.

# Project Structure
calculator-boilerplate
│   app.js
│   package.json
│
├───routes
│       calculator.js
│
└───views
        error.html
        index.html
        result.html

# How To Run
To install dependencies: 
  npm i
To run: 
  node app.js
